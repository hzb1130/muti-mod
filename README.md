基于某个不知名自改mod，重置了部分功能。

## 📊 一、信息显示（HUD）

| 功能 | 说明 | 默认值 |
|------|------|--------|
| 开启信息显示 | 总开关，控制所有HUD显示 | `false` |
| 字体大小 | 调整HUD文字大小 | 15-30 (默认20) |
| 显示天气 | 显示天气类型、风力、温度、倒计时 | `true` |
| 天气X | 天气面板X坐标 | 0-2000 (默认20) |
| 天气Y | 天气面板Y坐标 | 0-1200 (默认20) |
| 玩家信息 | 显示移速、维生素C | `true` |
| 玩家X | 玩家面板X坐标 | 0-2000 (默认20) |
| 玩家Y | 玩家面板Y坐标 | 0-1200 (默认60) |
| 猎杀信息 | 显示各类猎物击杀统计 | `true` |
| 猎杀X | 猎杀面板X坐标 | 0-2000 (默认20) |
| 猎杀Y | 猎杀面板Y坐标 | 0-1200 (默认100) |

### 显示内容示例
- **天气**：`生存 X天  HH:MM  天气类型  风力  X°C  天气剩余:HH:MM`
- **玩家**：`移速: X.X m/s  维C: XX%`
- **猎杀**：`猎物: 兔X 松鸡X 鹿X 狼X 熊X 驼鹿X 美洲狮X`

## ⏰ 二、时间控制

| 功能 | 说明 | 默认值 |
|------|------|--------|
| 全局变速并显示 | 启用时间倍率控制 | `false` |
| 时间倍率 | 游戏速度倍率 | 0.1x - 10x (默认1x) |
| 时间热键 | 切换变速的热键 | `Keypad8` |

> **使用说明**：按下热键后游戏时间按设定倍率加速/减速，再次按下恢复正常速度

---

## 🕊️ 三、飞行模式

| 功能 | 说明 | 默认值 |
|------|------|--------|
| 一键飞行 | 启用飞行模式 | `false` |
| 飞行热键 | 进入飞行模式的热键 | `Keypad7` |

> **使用说明**：按下热键后进入自由飞行模式，可穿越地形和建筑

---

## ⚔️ 四、战斗

| 功能 | 说明 | 默认值 |
|------|------|--------|
| 始终显示准星 | 任何时候都显示瞄准准星 | `false` |
| 伤害倍率 | 对野生动物造成的伤害倍率 | 0.1x - 10x (默认1x) |
| 击杀播报 | 击杀时屏幕显示通知 | `false` |

### 击杀播报效果
- 屏幕淡黄色提示：`击杀 ☠ 动物名称`
- 支持：狼、熊、驼鹿、鹿、兔子、美洲狮、松鸡

## 🔥 八、生火

| 功能 | 说明 | 默认值 |

---

## 🍳 十、烹饪

| 功能 | 说明 | 默认值 |
|------|------|--------|
| 不烧焦食物 | 食物不会烧焦，水不会烧干 | `false` |

### 效果说明
- 食物煮熟后永久保持在「已煮熟」状态
- 不会进入「烧焦」状态
- 烧水不会烧干
